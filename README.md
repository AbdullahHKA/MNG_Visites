#  Healthcare Visits Dashboard – National Guard Health Affairs

##  Overview
This project is an interactive **Power BI dashboard** that analyzes healthcare visits for **National Guard Health Affairs** using **open data from the Saudi National Open Data Platform**.

The dashboard provides insights into healthcare service utilization through interactive reports, KPIs, and trend analysis to support data-driven decision-making.

---

##  Objectives
- Analyze healthcare visits across facilities.
- Monitor visit trends over time.
- Compare healthcare facilities.
- Identify the contribution of each facility to total visits.
- Present healthcare data through interactive visualizations.

---

##  Dashboard Features
- Total healthcare visits
- Visits by healthcare facility
- Visits by city
- Interactive filters (Slicers)
- KPI Cards

---

##  Tools & Technologies
- Power BI
- DAX
- Power Query
- Microsoft Excel

---



##  Data Source
Saudi National Open Data Platform (Open Data).
